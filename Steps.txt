Create EC2 Instance from AWS Console.
a. Create new key pair to make connection between Putty and Sever.

Pick Public IP address from EC2 instance and provide in Putty Host.
Go To SSH -- AUTH -- path of ..pk file.
Go To DATA, give ec2-user as user name and open.
